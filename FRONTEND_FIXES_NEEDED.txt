=== API CALLS FOR EACH PAGE ===

Dashboard (/admin/dashboard):
  - GET /api/v1/admin/dashboard/stats
  - Display: total transactions, amounts, users, accounts

Transactions (/admin/transactions):
  - GET /api/v1/transactions
  - Display: list of all transactions with filters

Accounts (/admin/accounts):
  - GET /api/v1/admin/accounts
  - POST /api/v1/admin/accounts/create (for create button)
  - Display: list of accounts with balances

Projects (/admin/projects):
  - GET /api/v1/projects
  - POST /api/v1/projects (auto-creates account!)
  - Display: list of projects with progress

Reports (/admin/reports):
  - GET /api/v1/admin/reports/financial
  - GET /api/v1/admin/reports/monthly-summary
  - Display: financial reports and charts

Settings (/admin/settings):
  - Your custom settings logic
  - User profile updates, etc.
